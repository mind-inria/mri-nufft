.. _tutorials:

Tutorials
=========

A guided, step-by-step introduction to non-Cartesian MRI reconstruction with
MRI-NUFFT, aimed at newcomers to the field. Each tutorial motivates its
tools by first showing what goes wrong without them, before introducing the
fix -- read them in order, as each one builds on the previous.

If you already know what you're looking for, see the :ref:`general_examples`
gallery instead for focused how-to snippets.
